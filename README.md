# Tic-Tac-Toe

A simple implementation of the game Tic-Tac-Toe in Java.
The game is available in two modes: single-player and two-player.

A simple run of the game can look like this:
```
Main Menu
1. One Player
2. Two Player
3. Quit
Selection: 
1
Player 1 - Select Between 'X' or 'O': 
X
   |   |   
---+---+---
   |   |   
---+---+---
   |   |   

Player 1 - Select row and column: 
1 1
 X |   |   
---+---+---
   |   |   
---+---+---
   |   |   

 X | O |   
---+---+---
   |   |   
---+---+---
   |   |   

Player 1 - Select row and column: 
2 1
 X | O |   
---+---+---
 X |   |   
---+---+---
   |   |   

 X | O | O 
---+---+---
 X |   |   
---+---+---
   |   |   

Player 1 - Select row and column: 
3 1
 X | O | O 
---+---+---
 X |   |   
---+---+---
 X |   |   

Player 1 wins
```
